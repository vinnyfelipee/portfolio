Sobre mim

Meu nome é Vinny Felipe, um estudante dedicado de Análise e Desenvolvimento de Sistemas com uma paixão avassaladora pela tecnologia. Minha jornada acadêmica tem sido focada em adquirir habilidades abrangentes, desde o desenvolvimento web com Python, JavaScript, HTML e CSS até a criação de visualizações de dados poderosas no Power BI. Com um olhar voltado para o futuro, meu objetivo é me tornar um desenvolvedor fullstack, integrando conhecimentos em qualidade de software (Q.A) para assegurar que cada projeto atenda aos mais elevados padrões de excelência. Além do meu comprometimento com a tecnologia, minha afinidade por música e jogos adiciona uma dimensão criativa ao meu perfil, enriquecendo minha abordagem holística para resolver desafios inovadores.

Estou animado para contribuir com minha paixão, criatividade e habilidades técnicas para projetos empolgantes. Se você está em busca de um profissional dinâmico e em constante evolução, estou pronto para enfrentar novos desafios e contribuir significativamente para o desenvolvimento de sistemas. Estou ansioso para somar de todas as maneiras ao meu alcance!
